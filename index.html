Wireless and Mobile Computing

What is Wireless and Mobile Computing

Wireless refers to the method of transferring information between a computing device such as a personal data assistant PDA and the data source such as an agency database server without a physical connection. Not all wireless communication technology are mobile. Mobile simply describes a computing device that is not restricted to a device. A mobile device may be a PDA, a smart cell phone or web phone, a laptop computer or any one of the numerous other devices that allow the user to complete computing tasks without being connected to a network. Wireless communication is simply data communication without the use of landlines.

WLL or WiLL wireless in the local loop is meant to serve subscribers at houses or offices. In WLLl services the telephone provided is expected to be as good as wired phone

SMS

Short message service is the transmission of short text messages to and from the mobile phone. Text messages must not be longer than some fixed number of alphanumeric characters and should contain no image or graphics.

Voicemail

The voicemail refers to an email system that supports audio. User can leave spoken message for one another and listen to the messages by executing the appropriate command in the email system
